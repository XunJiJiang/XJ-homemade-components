## プロジェクトのセットアップ

```shell
npm install
```

### 開発用のコンパイルとホットリロード

```shell
npm run serve
```

### 実稼働用にコンパイルおよび縮小します

```shell
npm run build
```

## について

これはテスト用の自作コンポーネントです

ファイル構成に不正がある可能性があります。

現在サポートされているのは以下のみです:

- ダイアログ (一部の関数は element を模倣します)
- notification  (関数は element を模倣します)
- message (スタイルと機能を模倣した element)
- コンソール出力
